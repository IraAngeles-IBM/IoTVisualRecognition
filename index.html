IoT Fire Alarm using Node-RED and IBM Visual Recognition

This project is a how-to guide on using Node-RD and IBM Visual Recognition service to analyse an image and based on a trigger word then send a event command to IoT Device. In this example node-red is used to create an image upload form which sends the image to IBM Visual Recognition and trigger an iot device whenever fire is detected in the image.

  1. Login/signup in to your IBM Cloud Account
  2. Goto Catalog and create the following services:
  3. Install Node-RED NOTE : Please follow the tutorial to install Node-RED in IBM Cloud

  4. Create IBM Cloud Service Internet of Things Platform NOTE : You need to create the A Platform Starter App,not Internet of things Platform service.I suggest you use the link above to quickly avoid confusion.

  5. Create IBM AI Visual Recognition

NOTE: You can skip steps 3,4 & 9

  1. Goto connections tab and open IotF Service and Launch Watson Iot Platform
  2. Navigate to Devices tab and Add Device
  3. Once the Starter app is running,visit app url and configure Node-red
  4. Login to your node-red flows dashboard and import flow from flows.json
  5. Create new visual recognition credentials
  6. Update api key in the visual recognition node-red flow
  7. Update the device details in IBM Iot device node {Optional}
  8. Visit /upload and upload required image
  9. Change the string inside includes from trigger action function to set a different trigger
